# 贡献指南 / Contributing Guide

感谢你对 tiangou-quotes 的关注！欢迎提交新的舔狗语录、情话金句和毒鸡汤。

## 如何贡献语录

### 1. 提交 Issue（推荐新手）

如果你有好的语录但不想动手改代码，直接 [开一个 Issue](../../issues/new) 即可：

**标题格式**：`[新增语录] 分类名称`

**内容模板**：
```
## 语录内容
> "你的语录内容"

## 建议分类
- [ ] 被删好友/被拉黑
- [ ] 被骂/被怼
- [ ] 表白被拒
- [ ] 被忽视/不回消息
- [ ] 卑微送礼物
- [ ] 自我安慰/正面解读
- [ ] 搞笑套路
- [ ] 暖心表白
- [ ] 毒鸡汤/人间清醒
- [ ] 简短金句

## 备注（可选）
来源、背景故事等
```

### 2. 直接提交 PR（推荐）

1. Fork 本仓库
2. 编辑 `tiangou-quotes/SKILL.md`
3. 在对应分类下追加新语录，编号顺延
4. 提交 PR，描述改动内容

**PR 标题格式**：`feat: 新增 X 条「分类名」语录`

**注意事项**：
- 每条语录独占一行，以数字编号开头
- 避免与已有语录高度重复
- 保持语录的趣味性和传播性
- 不要删除或修改已有语录（除非有明显错误）

### 3. 语录质量标准

✅ **欢迎**：
- 有趣、有梗、有传播力的语录
- 覆盖新的场景或情绪
- 经典名句的新版本

❌ **不欢迎**：
- 与已有语录高度重复的内容
- 含有恶意攻击、歧视性内容的语录
- 无意义的灌水内容

## 新增分类

如果现有 10 个分类无法覆盖你的语录，可以在 Issue 中提议新增分类。请说明：
- 分类名称
- 适用场景描述
- 至少 5 条该分类的语录

## 代码风格

- SKILL.md 使用标准 YAML frontmatter + Markdown
- 语录文本保持中文原汁原味
- 编号使用阿拉伯数字
